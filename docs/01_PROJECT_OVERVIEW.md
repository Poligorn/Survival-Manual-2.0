# Обзор проекта: Survival Wiki

## Описание
**Survival Wiki** — это оффлайн-приложение (справочник по выживанию) для Android, переведенное на русский язык на основе открытого источника: [SurvivalManual Wiki](https://github.com/ligi/SurvivalManual/wiki).

Цель приложения — предоставить пользователю быстрый и надежный доступ к критически важной информации по выживанию в дикой природе и экстремальных ситуациях, даже при полном отсутствии подключения к интернету.

## Ключевые особенности
1. **Полная оффлайн-доступность:** Вся база статей, изображений и схем "зашита" в приложение. Не требуется загрузка из сети.
2. **Темная тема оформления:** Удобна для использования в условиях низкой освещенности и экономит заряд батареи.
3. **Быстрый поиск и теги:** Позволяет мгновенно находить нужные инструкции по ключевым словам или категориям (например: #первая_помощь, #узлы, #разведение_огня).
4. **Полнотекстовый поиск в статьях:** Возможность искать нужную информацию не только по заголовкам, но и по всему тексту статей.
5. **Иллюстрации с комментариями:** Наличие внутри статей подробных изображений, схем и графиков с поясняющими подписями для лучшего понимания инструкций.
6. **История просмотров:** Блок "Недавно читали" на главном экране обеспечивает быстрый возврат к актуальным статьям.

## Архитектурные требования
- Платформа: Android.
- Локальная база данных (Room) или встроенные JSON/Markdown файлы с предзагруженным переведенным контентом.
- Использование современных паттернов разработки: MVVM/MVI, Jetpack Compose.

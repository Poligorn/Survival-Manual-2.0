# Технический стек и архитектура (Android)

Приложение будет разрабатываться с использованием современных инструментов Android-разработки.

## Технологии

- **Язык программирования:** Kotlin.
- **UI Фреймворк:** Jetpack Compose (Material Design 3). Обеспечивает быструю и гибкую разработку красивых интерфейсов, включая сложную темную тему.
- **Архитектура:** MVVM (Model-View-ViewModel) + Clean Architecture (опционально, в зависимости от масштабов).
- **Навигация:** Jetpack Navigation Compose (Navigation Bar, навигация по графу экранов).
- **База данных:** Room (SQLite) для локального хранения структуры статей, тегов и истории просмотров.
  - **Для полнотекстового поиска (Full-Text Search):** Будет использоваться механизм **FTS4** встроенный в Room. Это позволит осуществлять мгновенный и эффективный поиск слов и фраз по всему содержимому статей, а не только по заголовкам.
- **Парсинг данных:** Moshi или Kotlinx.serialization (для чтения JSON из assets при первом запуске или миграции данных).
- **Отображение контента и медиа:** 
  - Markdown-парсер для Compose (например, `compose-markdown` или решение на основе кастомных блоков) для рендеринга текста, списков и заголовков.
  - **Coil** для загрузки и отображения локальных изображений прямо из папки `assets`. Для каждого изображения будет предусмотрен UI-блок подписи (caption) под картинкой, стилизованный для удобного чтения.

## Поток данных (Data Flow)

1. При первом запуске приложение считывает JSON файлы (или Markdown) из папки `assets`.
2. Данные сохраняются в локальную базу данных (Room), создаются FTS-индексы для поиска по контенту.
3. При вводе запроса в строку поиска происходит запрос к таблице FTS в SQLite, возвращающий список статей с совпадениями.
4. Каждое открытие статьи обновляет поле `last_read_timestamp` в таблице истории для отображения в блоке "Недавно читали".

## Дизайн

Цветовая палитра основана на референсном изображении:

- **Фон экранов:** Глубокий черный (`#000000`) или темно-серый (`#121212`).
- **Фон карточек (CardView):** Темно-серый (`#1E1E1E` или `#2C2C2C`), с легким скруглением углов (Corner Radius ~ 16dp).
- **Акцентный цвет:** Оранжевый (`#FF8C00` или `#FF6D00`) — используется для иконки "Главная", логотипа и активных элементов.
- **Текст:** Белый (для заголовков), светло-серый (для основного текста статей, подзаголовков, времени и подписей к изображениям).

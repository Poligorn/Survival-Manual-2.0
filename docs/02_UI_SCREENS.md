# Пользовательский интерфейс (UI) и Навигация

Приложение состоит из 3 основных экранов и отдельного экрана чтения статьи. Навигация между основными экранами осуществляется через **Bottom Navigation Bar** (Нижнюю панель навигации).

## Нижняя панель (Bottom Navigation Bar)
Содержит 3 пункта:
1. **Главная (Home)** — Иконка домика. (Активный цвет: Оранжевый).
2. **Каталог (Catalog)** — Иконка раскрытой книги.
3. **Настройки (Settings)** — Иконка шестеренки.

---

## 1. Главный экран (Home)
Это экран, который пользователь видит при запуске приложения.

**Структура экрана (сверху вниз):**
- **Шапка (Header):**
  - Фоновое изображение (темный лес, атмосферная картинка).
  - Логотип приложения (иконка огня/пламени) и заголовок `Survival Wiki`.
- **Блок поиска:**
  - Строка поиска с плейсхолдером `Поиск по базе...` и иконкой лупы.
  - *Поиск осуществляется как по заголовкам статей, так и по их содержимому (полнотекстовый поиск).* При вводе текста открывается экран с результатами поиска (Search Results), где подсвечиваются найденные совпадения в тексте.
- **Карусель тегов (Chips):**
  - Горизонтальный список популярных или недавних тегов: `#первая_помощь`, `#узлы`, `#разведение_огня` и т.д.
- **Секция "Недавно читали" (Recently read):**
  - Заголовок `Недавно читали`.
  - Вертикальный список карточек (CardView) с историей просмотров.
  - Каждая карточка содержит:
    - Тематическую иконку (капля воды, костер, спасательный инструмент).
    - Заголовок статьи (например, `Очистка воды в полевых условиях`, `Спасательные узлы`).
    - Подзаголовок (время просмотра: `Открыто 2 часа назад`).

---

## 2. Экран "Каталог" (Catalog)
Служит для структурированного просмотра всех доступных статей по категориям.

**Структура экрана:**
- **Заголовок:** `Каталог` или `Все разделы`.
- **Список категорий (Грид или Список):**
  - Психология выживания
  - Базовые потребности (Медицина, Укрытие, Огонь, Вода)
  - Навигация и ориентирование
  - Инструменты и узлы
  - Растения и животные
- При клике на категорию открывается вложенный список статей.

---

## 3. Экран чтения статьи (Article Detail Screen)
Экран, который открывается при выборе статьи из каталога, результатов поиска или "недавно читали".

**Структура экрана:**
- **Навигация:** Кнопка "Назад" (на верхней панели).
- **Заголовок статьи.**
- **Теги статьи** (кликабельные).
- **Контент статьи:** 
  - Форматированный текст (абзацы, списки, заголовки).
  - **Изображения с комментариями:** Встроенные в текст иллюстрации и схемы. Под каждым изображением находится текстовая подпись (комментарий), объясняющая суть картинки (обычно выделяется курсивом и немного меньшим шрифтом).

---

## 4. Экран "Настройки" (Settings)
Экран управления приложением.

**Структура экрана:**
- **Заголовок:** `Настройки`.
- **Опции:**
  - Очистка истории просмотров (кнопка "Удалить недавние").
  - Выбор темы (Системная / Темная / Светлая). *По умолчанию: Темная*.
  - Управление размером шрифта для чтения статей (Мелкий / Средний / Крупный).
- **О приложении (About):**
  - Версия приложения.
